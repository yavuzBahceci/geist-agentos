# Standard: Enriched Knowledge Templates

## Overview

This document defines the templates for files in `geist/config/enriched-knowledge/`. These files are generated by research workflows and used during agent deployment for better specialization.

---

## Directory Structure

```
geist/config/enriched-knowledge/
├── README.md                  # Index and summary
├── library-research.md        # Per-library best practices
├── stack-best-practices.md    # Architecture patterns
├── domain-knowledge.md        # Industry-specific patterns
├── version-analysis.md        # Dependency version status
└── security-notes.md          # Security vulnerabilities
```

---
